Get Started with Orus

1. Install

curl -sSf https://orus-lang.org/install.sh | sh

Install Orus with our simple one-line installer

2. Create

orus new my-project
cd my-project

Create a new Orus project and navigate to it

3. Run

orus run

Compile and run your Orus application
